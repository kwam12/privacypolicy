Face Data and Photo Analysis

1. What face data does the app collect?

The app collects a user-submitted photo that includes facial data for the purpose of skin analysis. This photo data is temporarily used for processing skin-related metrics and generating a skincare score.

2. Use of Face Data

The face data (user-submitted photo) is solely used to analyze skin health, specifically for identifying potential skin issues such as acne, dryness, or uneven tone, and to generate a personalized skincare score. This analysis is done in real time and only for the user's immediate viewing.

3. Data Sharing and Third-Party Practices

The face data is transmitted to OpenAI's ChatGPT API for processing, where the skin analysis score is generated. Here are the specific details about third-party handling of your face data:

OpenAI (ChatGPT)

  • Data Storage: OpenAI temporarily stores face data during processing for a maximum of 30 days.
  • Purpose of Storage: This temporary storage is necessary for:
    • Service improvement and training
    • Debugging and error resolution
    • Compliance with legal requirements
  • Storage Duration Justification: The 30-day retention period allows OpenAI to:
    • Address any technical issues that may arise
    • Respond to user inquiries or disputes
    • Maintain service quality and accuracy
  • Data Protection: OpenAI implements industry-standard security measures to protect all processed data, including encryption and secure data centers.
  • Data Usage: OpenAI's use of the data is governed by their privacy policy, which can be found at https://openai.com/privacy

4. Data Retention in Our App

The app retains photos locally on the user's device only if the user explicitly chooses to save them for tracking their progress over time. Users can delete their locally stored photos at any time through the app's settings.

5. User Controls and Rights

Users have the following rights regarding their face data:

  • The right to delete locally stored photos at any time
  • The right to opt out of progress tracking
  • The right to request information about their data usage
  • The right to withdraw consent for data processing

6. Data Security Measures

We implement several security measures to protect your face data during transmission and processing:

  • End-to-end encryption during data transmission
  • Secure HTTPS protocols
  • Regular security audits and updates
  • Restricted access to user data
